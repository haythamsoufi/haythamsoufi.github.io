# Feature Screenshots

Place your feature screenshots in this folder with the following names:

1. **form-builder.jpg** - Screenshot of the Form Builder interface
2. **multi-language.jpg** - Screenshot showing multi-language support
3. **analytics.jpg** - Screenshot of the Analytics dashboard
4. **mobile.jpg** - Screenshot of the mobile app interface
5. **security.jpg** - Screenshot of security settings/features
6. **communication.jpg** - Screenshot of the communication system

## Image Requirements

- **Format**: JPG or PNG
- **Recommended size**: 1920x1200px (16:10 aspect ratio for laptop mockup)
- **Aspect ratio**: 16:10 (will be displayed in laptop mockup frame)

The images will automatically display in the laptop mockup frames. If an image is missing, a placeholder will be shown instead.
