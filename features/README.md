## Image Requirements

### Laptop Mockups (backoffice.jpg, public-website.jpg, form-builder.jpg, multi-language.jpg, analytics.jpg, mobile.jpg, security.jpg, communication.jpg)

- **Format**: JPG or PNG
- **Recommended size**: 1920x1200px (16:10 aspect ratio)
- **Aspect ratio**: 16:10 (will be displayed in laptop mockup frame)

### Phone Mockup (mobile-app.jpg)

- **Format**: JPG or PNG
- **Recommended size**: 1080x2340px (9:19.5 aspect ratio, typical modern phone)
- **Aspect ratio**: 9:19.5 (will be displayed in phone mockup frame)

## Notes

- The images will automatically display in their respective mockup frames (laptop or phone)
- If an image is missing, a placeholder will be shown instead
- All images should be high-quality screenshots that clearly demonstrate the feature or interface
