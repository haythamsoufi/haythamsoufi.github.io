## Updates Made

### 1. Added Dynamic Indicators Feature ✨
- **Location**: Backoffice section, Screen 3
- **Description**: Kobo-style repeat groups allowing focal points to dynamically add indicators from the indicator bank
- **Key Features Highlighted**:
  - Real-time searchable indicator selection
  - Advanced filtering (type, unit, emergency status, sector)
  - Full disaggregation support (sex, age, sex+age)
  - Data availability flags
  - Custom labels for context-specific reporting
  - Support for all indicator types (number, percentage, yes/no, text)

### 2. Added Entity-Based Email Campaigns Feature ✨
- **Location**: Backoffice section, Screen 6
- **Description**: Targeted email campaigns to organizational entities with intelligent To/CC distribution
- **Key Features Highlighted**:
  - Multi-level entity support (9 entity types)
  - Smart contact categorization by domain and role
  - Configurable To/CC distribution rules
  - Deduplication and batch sending
  - Campaign management and tracking

### 3. Added AI Chat System & Document Processing Feature ✨
- **Location**: Backoffice section, Screen 7
- **Description**: Advanced AI-powered chat system with RAG (Retrieval-Augmented Generation) capabilities
- **Key Features Highlighted**:
  - Intelligent chat assistant with multi-step reasoning (ReAct pattern)
  - 10+ tools for database queries, document search, country comparison
  - Document embedding and processing (PDF, Word, Excel, Text, Markdown, HTML)
  - Vector embeddings with pgvector (OpenAI + local fallback)
  - Hybrid search (vector similarity + keyword search)
  - Document Q&A with source citations
  - Reasoning traces and cost management
  - WebSocket streaming for real-time responses
