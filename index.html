Unified Data Collection System

A Modern Platform for Humanitarian Data Management

The Unified Data Collection System is a comprehensive, open-source platform designed to streamline data collection, management, and reporting for humanitarian organizations. Built with the needs of NGOs, international organizations, and field teams in mind, this system enables teams to work more efficiently, make data-driven decisions, and improve collaboration across organizational levels.

Key Features

üé® Powerful Form Builder

Create sophisticated data collection forms without programming. Features conditional logic, dynamic sections, multi-language support, and validation rules.

üåç Multi-Language Support

Available in 7 languages (English, French, Spanish, Arabic, Chinese, Russian, Hindi) with full right-to-left support for Arabic.

üìä Advanced Analytics

Comprehensive analytics including disaggregation analysis, indicator tracking, and customizable reports for data-driven decision making.

üì± Mobile Access

Native mobile apps for iOS and Android with offline support, so teams can work even without internet connectivity.

üîê Enterprise Security

Enterprise-grade security with role-based access control and optional Azure AD integration for single sign-on.

üí¨ Smart Communication

Intelligent notification system and email campaigns that reach the right people at the right organizational level.
