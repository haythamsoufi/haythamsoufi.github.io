# Haytham Alsoufi - Portfolio

Jekyll-based portfolio website hosted on GitHub Pages.

## Setup

1. Install Jekyll:
   ```bash
   gem install bundler jekyll
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Visit: http://localhost:4000

## Deployment

This site is automatically deployed to GitHub Pages when you push to the `main` branch.

## Customization

- Edit `_config.yml` for site settings
- Edit markdown files in root for page content
- Customize `_layouts/` for page layouts
- Add CSS in `_sass/` or inline in layouts
